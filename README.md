# 2.5V DC Power Supply

## Overview

This project was developed as part of our CEP 2nd Semester Project.
The aim was to design a regulated 2.5V DC power supply using basic electronic components.

## Components Used

* 12V Step-Down Transformer
* 4 × 1N4007 Diodes
* 100µF Capacitor
* 1kΩ Resistor
* 2.5V Zener Diode
* 5kΩ Load Resistor

## Working

The transformer steps down AC voltage to 12V AC.
A bridge rectifier converts AC to DC, while the capacitor filters the output.
The Zener diode regulates the voltage to provide a stable 2.5V DC output.

## Output

Stable 2.5V DC Output

## Learning Outcomes

* AC to DC Conversion
* Rectification and Filtering
* Voltage Regulation using Zener Diode
* Practical Circuit Implementation
